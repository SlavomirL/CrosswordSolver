Are you stuck in some kind of words game?

Try out one of these cheating apps:

Crossword Solver
DESCRIPTION:

You can enter letters as you see them in your crossword. Fill only known letters on correct positions and Crossword Solver will find all words that would fit in.

Fixed Length Word Builder
DESCRIPTION:

You can enter all letters that are available for you in a Scrabble-type game. Fixed Word Builder will generate all possible words built by using every letter once.

Flexible Length Word Builder
DESCRIPTION:

You can enter all letters that are available for you in a Scrabble-type game. Flex Word Builder will generate all possible words from given letters. Result will contain shorter words as well as words where all letters are used.
